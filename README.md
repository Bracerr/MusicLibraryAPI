# SongLibrary REST API (Go & Echo Framework)

## Технологии
- **Go**: 1.23.1

## Запуск приложения
### Docker
```bash
docker-compose up --build
```

### Local
```bash
go mod tidy
```
```bash
docker-compose up -d db
```
```bash
cd src/cmd
```
```bash
go run main.go
```
